## Check Test 　 p181

### Q1 色（color）が赤で、インチ（inch）がの、自転車オブジェクト bicycle を表現するコードを書いてください。

A.

    const bicycle = {
    	color: 'red',
    	inch: 25
    }

### Q2 Q1 で書いたオブジェクトから、色（color）の値を取得するコードを書いてください。

A.

    console.log(bicycle.color);

### Q3 Q1 で書いたオブジェクトのインチ（inch）を 16 に変更するコードを書いてください。

A.

    bicycle.inch = 16;
